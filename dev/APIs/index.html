FractionalDiffEq.FDEProblem — Type
FDEProblem(f, α, u0, T, h)

Define a Fractional Differential in time interval [0, T] with initial value y(0)=u₀, α-order derivative and step size h.

Detailed types

FractionalDiffEq.PECE — Type

Predict-Evaluate-Correct-Evaluate algorithm.

For more details, please refer to Predictor-Corrector algorithms

This PECE algorithm is taken from Diethelm's paper.

@article{
title={A predictor-corrector approach for the numerical solution of fractional differential equations},
author={Diethelm, Kai and Ford, Neville J. and Freed, Alan D.}
doi={https://doi.org/10.1023/A:1016592219341}
}
source

Using triangular strip matrices to discrete fractional ordinary differential equations to simple algebra system and solve the system.

Some models

FractionalDiffEq.bagleytorvik — Function
bagleytorvik(p1, p2, p3, T, h)

By specifying the parameters of Bagley Torvik Equation, we can use bagleytorvik to directly obtain the numerical approximation of a bagley torvik equation.

p2 ≠ 0

Please note that the parameter of fractional derivative part must not be 0

source
FractionalDiffEq.diffusion — Function
diffusion(α, β)

Diffusion equation with time fractional derivative.

α and β are the coefficients of the time derivative and spatial derivative.
